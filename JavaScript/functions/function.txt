Function:

A Function is a block of code that takes an input(optional), process it and gives a result.

Without function we cannot write a single program using JS, if we want integrate with UI.

Syntax:

function function_name(parameters)
{
    //code

    //return data;
}



So, there are four ways to represent a function.


1. Function with parameters and with return type.

    eg: function add(a,b){ // a,b are parameters

        var c = a+b;

        return c;
    }
    add(10,20);


2. Function without parameters and with return type.

    eg: function getScore(){
        return "10/1";
    }

3. Function with parameters and without return type.

    eg: function login(otp){
        if(otp==1023){
            console.log("logged in");
        }
        else{
            console.log("invalid otp");
        }

    }

4. Function without parameters and without return type.

     eg: function getCountries(){
         ["India","Australia"];

     }

     function displayCountries(){

         var countries = getCountries();

     }
